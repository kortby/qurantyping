## Quelles données collectons-nous ?

QuranTyping collecte les données suivantes :

*   Email
*   Nom d'utilisateur
*   Informations sur chaque test de frappe
*   Vos paramètres actuellement actifs
*   Combien de tests de frappe vous avez commencés et terminés
*   Combien de temps vous avez tapé sur le site Web

QuranTyping ne collecte PAS :

*   textes personnalisés (ils sont stockés dans le stockage local de votre navigateur)

Si vous pensez qu'un certain type de données manque dans les listes ci-dessus, n'hésitez pas à nous contacter et nous répondrons à toutes les questions et mettrons à jour la politique de confidentialité.

## Comment collectons-nous vos données ?

Vous fournissez directement la plupart des données que nous collectons. Nous collectons des données et traitons des données lorsque vous :

*   Créez un compte
*   Complétez un test de frappe
*   Modifiez les paramètres sur le site Web

## Comment utiliserons-nous vos données ?

QuranTyping collecte vos données afin que nous puissions :

*   Vous permettre de consulter l'historique des résultats des tests précédents que vous avez effectués
*   Enregistrer les résultats des tests que vous passez et vous montrer des statistiques basées sur ceux-ci
*   Mémoriser vos paramètres
*   Afficher les classements

Si vous êtes trouvé en train de tricher ou d'exploiter le site Web, nous pouvons stocker des versions hachées de votre nom d'utilisateur et de votre email pour vous empêcher de créer de nouveaux comptes.

## Comment stockons-nous vos données ?

QuranTyping stocke vos données en toute sécurité en utilisant MySQL.

## Quels sont vos droits en matière de protection des données ?

QuranTyping souhaite s'assurer que vous êtes pleinement conscient de tous vos droits en matière de protection des données. Chaque utilisateur a droit aux éléments suivants :

*   **Le droit d'accès** – Vous avez le droit de demander à QuranTyping des copies de vos données personnelles. Nous pouvons limiter le nombre de fois que cette demande peut être faite en fonction de la taille de la demande.
*   **Le droit de rectification** – Vous avez le droit de demander à QuranTyping de corriger toute information que vous jugez inexacte. Vous avez également le droit de demander à QuranTyping de compléter les informations que vous jugez incomplètes.
*   **Le droit à l'effacement** – Vous avez le droit de demander à QuranTyping d'effacer vos données personnelles, sous certaines conditions. (Les données hachées mentionnées dans la section "Comment utiliserons-nous vos données ?" ne seront pas supprimées, car elles sont essentielles pour empêcher l'exploitation du site Web)
*   **Le droit de restreindre le traitement** – Vous avez le droit de demander à QuranTyping de restreindre le traitement de vos données personnelles, sous certaines conditions.
*   **Le droit de s'opposer au traitement** – Vous avez le droit de vous opposer au traitement de vos données personnelles par QuranTyping, sous certaines conditions.
*   **Le droit à la portabilité des données** – Vous avez le droit de demander à QuranTyping de transférer les données que nous avons collectées à une autre organisation, ou directement à vous, sous certaines conditions.

## Analytique

Comme la plupart des sites Web, nous utilisons Google Analytics, qui collecte des informations que votre navigateur envoie chaque fois que vous visitez le site Web. Ces données peuvent inclure des adresses de protocole Internet (IP), le type de navigateur, le fournisseur de services Internet (FAI), l'horodatage, les pages de référence/sortie et le temps passé sur chaque page. CES DONNÉES NE CONTIENNENT AUCUNE INFORMATION PERSONNELLEMENT IDENTIFIABLE. Nous utilisons ces informations pour analyser les tendances, administrer le site, suivre les mouvements des utilisateurs sur le site Web et recueillir des informations démographiques.

Pour plus d'informations sur la politique de confidentialité de Google Analytics, veuillez visiter : [https://support.google.com/analytics/answer/6004245?hl=fr](https://support.google.com/analytics/answer/6004245?hl=fr)

## Que sont les cookies ?

Les cookies sont des fichiers texte placés sur votre ordinateur pour collecter des informations de journal Internet standard et des informations sur le comportement des visiteurs. Lorsque vous visitez nos sites Web, nous pouvons collecter automatiquement des informations vous concernant via des cookies ou une technologie similaire.

Pour plus d'informations, consultez Cookie HTTP sur Wikipédia.

## Comment utilisons-nous les cookies ?

QuranTyping utilise les cookies de différentes manières pour améliorer votre expérience sur notre site Web, notamment :

*   Vous garder connecté
*   Mémoriser vos paramètres actifs
*   Analyse du trafic
*   Fins publicitaires

## Quels types de cookies utilisons-nous ?

Il existe plusieurs types de cookies différents ; cependant, notre site Web utilise des cookies de fonctionnalité. QuranTyping utilise ces cookies pour vous reconnaître sur notre site Web et mémoriser vos paramètres précédemment sélectionnés.

## Comment gérer vos cookies

Vous pouvez configurer votre navigateur pour qu'il n'accepte pas les cookies, et le site Web ci-dessus vous indique comment supprimer les cookies de votre navigateur. Cependant, dans quelques cas, certaines fonctionnalités de notre site Web peuvent se comporter de manière inattendue ou ne pas fonctionner en conséquence.
